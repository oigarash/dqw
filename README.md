# Tools for Dragon Quest Walk
ドラクエウォークでちょっとした計算とか調査に必要なときに利用できるツール集

## パーティ行動順
パーティの行動順を確認できるツール。

### オンラインで使う
GitHub Pages で公開中: `https://<username>.github.io/<repository-name>/`

※ GitHub Pages を有効にするには：
1. GitHubのリポジトリページで「Settings」→「Pages」に移動
2. 「Source」で「Deploy from a branch」を選択
3. 「Branch」で `main` ブランチと `/docs` フォルダを選択
4. 「Save」をクリック
5. 数分後にページが公開されます
